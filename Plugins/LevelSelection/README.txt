# Level Selection Plugin Documentation

## Installation
1. Place the `LevelSelection` folder in your Unreal Engine plugin directory:
   - For project-specific installation: `[YourProjectFolder]/Plugins/`
   - For engine-wide installation: `[UnrealEngineInstallDir]/Engine/Plugins/`

2. Restart Unreal Engine.

## Usage
1. Once the plugin is installed, you will find a "Level Selection" button in the toolbar.
2. Click the button to open the Level Selection window.
3. Use the interface to organize your levels into categories for easier access.

## Compatibility
- Supported Unreal Engine versions: 5.0, 5.1, 5.2, 5.3, 5.4, 5.5.
- Platforms: Windows, Mac, Linux.

## Troubleshooting
- If the plugin does not appear after installation, ensure it is enabled in the Plugin Manager.
- Clear the Intermediate and Saved folders if you encounter issues.
